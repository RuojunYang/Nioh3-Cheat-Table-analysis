# 仁王 3 (Nioh 3) Cheat Table 说明文档

> **当前版本:** V 1.03.04  
> **适用平台:** Steam / PC  
> **主要功能:** 角色属性修改、数值逻辑注入

---

## 🛠 功能列表 / Features

### 1. 战斗核心修改
| 功能名称 | 技术实现 (Logic) | 详细说明 |
| :--- | :--- | :--- |
| **主角无敌** | `cmp [r11+148], 1` | 识别玩家 ID (`1`) 后锁定血量偏移 `[r11+20]`。 |
| **敌人 1 血** | `mov rbx, 1` | 非玩家单位受击即死，触发死亡判定。 |
| **攻防倍数** | `xmm15` 系数缩放 | 玩家受伤害减少，敌人受伤害增加；已处理 `0` 和 `NaN` 异常值。 |

### 2. 无限精力 (Infinity Ki)
* **攻击不消耗精力：** * 通过 `xorps xmm1, xmm1` 使攻击消耗量归零。
    * 特征判定：`[rdi+F0]` 为玩家标识。
* **受击不消耗精力：** * 独立函数处理，锁定受击时的精力减少量 `xmm6`。

### 3. 升级不消耗精华 (2026.3.7 更新)
* **升级不消耗精华：** * 通过跳过 `cmp rax,[rcx+10]` 判定使当前精华小于升级所需精华还可以升级。
